# Предложение дизайна для SAME SAME

## Концепция
**Уютный, спокойный, минималистичный стиль для интровертов**

## Цветовая палитра

### Основные цвета
- **Primary (Основной)**: `#8B7EC8` - Мягкий лавандовый/фиолетовый
  - Спокойный, но не скучный
  - Создает атмосферу доверия и уюта
  
- **Background (Фон)**: `#FAF9F6` - Теплый кремовый
  - Не белый, более уютный
  - Мягкий для глаз
  
- **Surface (Карточки)**: `#FFFFFF` - Белый
  - Чистый, но с мягкими тенями
  - Легкие границы для деликатности

### Акценты
- **Like (Лайк)**: `#7FD8BE` - Мягкий мятный
  - Позитивный, но не агрессивный
  - Ассоциируется с ростом и гармонией
  
- **Dislike (Дизлайк)**: `#FFB3BA` - Мягкий розовый
  - Не красный, более деликатный
  - Мягкое выражение несогласия
  
- **Highlight (Важное)**: `#FF6B9D` - Теплый коралловый
  - Для важных действий
  - Привлекает внимание, но не кричит

### Текст
- **Primary**: `#2D2D2D` - Мягкий темно-серый (не черный!)
- **Secondary**: `#6B6B6B` - Средний серый
- **Tertiary**: `#9B9B9B` - Светлый серый

## Особенности стиля

### Скругления
- **Большие радиусы**: 16-20px для карточек
- **Средние**: 12px для кнопок и полей ввода
- **Мягкие формы** создают уютную атмосферу

### Тени
- **Очень мягкие**: `opacity: 0.03-0.05`
- **Небольшой blur**: 10-20px
- **Легкое смещение**: 2-4px
- Создают глубину без агрессии

### Отступы
- **Просторные**: 16-24px между элементами
- **Не тесно**: много воздуха
- **Комфортное чтение**

### Типографика
- **Крупный шрифт**: легко читается
- **Высота строки**: 1.5 для комфорта
- **Letter spacing**: небольшой для элегантности

## Примеры использования

### Карточки
- Скругление: 20px
- Легкая граница: `#E8E6E3`
- Мягкая тень

### Кнопки
- Скругление: 16px
- Без резких теней
- Плавные переходы

### Прогресс-бары
- Мягкие цвета
- Плавные анимации
- Не агрессивные

## Психология цветов

### Почему эти цвета?
- **Лавандовый**: Спокойствие, креативность, духовность
- **Кремовый фон**: Уют, тепло, безопасность
- **Мятный**: Рост, гармония, свежесть
- **Мягкий розовый**: Нежность, деликатность

### Для интровертов
- Не яркие, не кричащие цвета
- Мягкие переходы
- Много пространства
- Спокойная атмосфера
